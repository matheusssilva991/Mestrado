# Fuzzy

Repositorio com estudos, tarefas e trabalhos da disciplina de logica fuzzy. Os
diretorios estao organizados por atividade, mantendo notebooks, codigos de apoio,
dados e resultados separados por contexto.

## Pastas

- [`docs/`](docs/): materiais de referencia da disciplina, incluindo livros e
  documentos auxiliares.
- [`tarefa_1/`](tarefa_1/): primeira tarefa pratica, com funcoes de pertinencia
  para idade e operadores fuzzy basicos.
- [`trabalho_1/`](trabalho_1/): sistema fuzzy para estimar risco de fadiga em
  turbinas, com notebook, script de execucao, base de regras, CSVs e figuras de
  saida.

## Trabalhos

| Pasta | Descricao | Entrada principal |
| --- | --- | --- |

| [`trabalho_1/`](trabalho_1/) | Modelo fuzzy completo para inferencia e defuzzificacao do risco de fadiga. | [`trabalho_1/src/main.ipynb`](trabalho_1/src/main.ipynb) |

Para detalhes de execucao, dependencias e saidas geradas, consulte o README de
